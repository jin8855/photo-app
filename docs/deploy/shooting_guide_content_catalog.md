# Shooting Guide Content Catalog (v1)

## 1. 목적

이 문서는 사전 촬영 가이드 기능에서 사용할 1차 콘텐츠 카탈로그다.

사전 촬영 가이드는 업로드 이미지나 분석 결과와 무관하게 동작한다. 사용자는 사진을 찍기 전에 카테고리와 기기 타입을 선택하고, 바로 적용 가능한 촬영 팁을 확인한다.

## 2. 1차 범위

카테고리:

- 인물 (`portrait`)
- 음식 (`food`)
- 풍경 (`landscape`)
- 야경 (`night`)
- 자동차/액션 (`automotive_action`)

기기 타입:

- Galaxy S25 Ultra (`galaxy_s25_ultra`)
- Galaxy S26 Ultra (`galaxy_s26_ultra`)
- 일반 스마트폰 (`generic_smartphone`)

출력:

- 조합별 3~5개 팁 카드
- 각 카드는 `title + body` 구조
- 검색 없음
- 즐겨찾기 없음
- DB 저장 없음
- 업로드 이미지 요구 없음
- 분석 결과 요구 없음

## 3. 콘텐츠 작성 규칙

모든 팁은 아래 기준을 따른다.

- 행동과 이유를 함께 쓴다.
- 바로 따라 할 수 있는 문장으로 쓴다.
- 추상적인 감성 표현보다 촬영 행동을 우선한다.
- 기기 스펙 설명보다 실제 촬영 상황을 우선한다.
- 너무 긴 설명을 피하고 1~2문장으로 정리한다.
- 분석 결과 기반 팁처럼 보이지 않게 쓴다.

## 4. 데이터 구조 예시

```ts
type ShootingGuideCategory =
  | "portrait"
  | "food"
  | "landscape"
  | "night"
  | "automotive_action";

type ShootingGuideDeviceType =
  | "galaxy_s25_ultra"
  | "galaxy_s26_ultra"
  | "generic_smartphone";

type ShootingGuideTip = {
  title: string;
  body: string;
};
```

## 5. Category: 인물 (`portrait`)

### Galaxy S25 Ultra (`galaxy_s25_ultra`)

- 얼굴은 화면 중앙에서 살짝 벗어나게 두세요. 너무 정중앙보다 자연스럽고 안정적인 구도가 됩니다.
- 실내에서는 노출을 조금 낮추세요. 피부 톤이 날아가지 않고 얼굴 윤곽이 더 또렷해집니다.
- 3배 줌으로 한두 걸음 떨어져 찍어보세요. 얼굴 왜곡이 줄고 배경이 부드럽게 정리됩니다.
- 강한 조명 아래에서는 정면보다 측면 빛을 활용하세요. 그림자가 깊어지고 얼굴 입체감이 살아납니다.

### Galaxy S26 Ultra (`galaxy_s26_ultra`)

- 얼굴은 화면 중앙에서 살짝 벗어나게 두세요. 인물이 덜 딱딱해 보이고 여백이 자연스럽게 생깁니다.
- 배경이 복잡하면 3배 줌으로 거리를 확보하세요. 얼굴은 선명하게, 배경은 정돈된 느낌으로 잡힙니다.
- 창가 빛을 옆에서 받게 두세요. 정면 조명보다 얼굴의 입체감이 부드럽게 살아납니다.
- 밝은 옷이나 흰 배경에서는 노출을 살짝 낮추세요. 피부와 배경 디테일이 더 안정적으로 남습니다.

### 일반 스마트폰 (`generic_smartphone`)

- 얼굴을 너무 가까이서 찍지 마세요. 한두 걸음 떨어지면 얼굴 왜곡이 줄어듭니다.
- 밝은 배경 앞에서는 인물이 어둡게 나올 수 있습니다. 얼굴 쪽을 눌러 초점과 밝기를 맞추세요.
- 정면보다 살짝 측면 각도로 찍으면 얼굴 윤곽이 더 자연스럽게 보입니다.
- 배경에 선이나 물건이 얼굴과 겹치지 않게 위치를 조금 바꿔보세요.

## 6. Category: 음식 (`food`)

### Galaxy S25 Ultra (`galaxy_s25_ultra`)

- 2~3배 줌으로 조금 떨어져 찍으세요. 음식 형태가 덜 왜곡되고 접시가 안정적으로 보입니다.
- 창가 자연광에서 찍으면 색감이 가장 자연스럽습니다. 노란 조명 아래에서는 음식 색이 탁해질 수 있습니다.
- 접시 전체보다 가장 맛있어 보이는 부분을 중심으로 잡으세요. 시선이 한곳에 모입니다.
- 윤기가 강한 음식은 노출을 살짝 낮추세요. 하이라이트가 날아가지 않아 더 맛있어 보입니다.

### Galaxy S26 Ultra (`galaxy_s26_ultra`)

- 2~3배 줌으로 찍으면 접시 가장자리 왜곡이 줄고 음식이 더 안정적으로 보입니다.
- 음식 위에서 바로 내려찍기보다 45도 각도로 찍어보세요. 높이와 질감이 더 잘 살아납니다.
- 창가 빛을 옆에서 받게 두면 음식의 결이 자연스럽게 드러납니다.
- 소스나 윤기가 있는 음식은 밝기를 조금 낮춰 디테일을 남기세요.

### 일반 스마트폰 (`generic_smartphone`)

- 음식 바로 앞에서 너무 가까이 찍기보다 팔을 조금 뻗어 거리를 확보하세요.
- 노란 실내 조명에서는 색이 탁해질 수 있습니다. 가능하면 창가나 밝은 조명 근처에서 찍으세요.
- 테이블 위 물건을 줄이면 음식이 더 깔끔하게 강조됩니다.
- 접시를 화면에 꽉 채우기보다 주변 여백을 조금 남기면 더 자연스럽습니다.

## 7. Category: 풍경 (`landscape`)

### Galaxy S25 Ultra (`galaxy_s25_ultra`)

- 초광각을 사용할 때는 수평선을 먼저 맞추세요. 풍경 사진은 수평이 안정감을 크게 좌우합니다.
- 하늘과 땅의 비율을 1:2 또는 2:1로 나눠보세요. 반반 구도보다 더 의도적으로 보입니다.
- 멀리 있는 산, 건물, 나무를 강조하고 싶다면 3배 줌을 사용하세요. 배경 압축감이 생깁니다.
- 밝은 하늘이 많을 때는 HDR을 켜세요. 하늘과 땅의 디테일을 함께 살리기 좋습니다.

### Galaxy S26 Ultra (`galaxy_s26_ultra`)

- 초광각으로 찍을 때 가장자리 왜곡을 확인하세요. 중요한 피사체는 화면 중앙 쪽에 두는 편이 좋습니다.
- 하늘이 예쁜 날에는 하늘을 2/3 정도로 크게 잡아 분위기를 살리세요.
- 특정 건물이나 산을 강조할 때는 3배 줌을 사용하세요. 멀리 있는 대상이 더 또렷하게 느껴집니다.
- 밝은 낮에는 노출을 살짝 낮추면 구름과 하늘 디테일이 더 잘 남습니다.

### 일반 스마트폰 (`generic_smartphone`)

- 먼저 수평선을 맞추세요. 풍경 사진은 수평만 맞아도 훨씬 안정적으로 보입니다.
- 하늘만 찍기보다 앞쪽 나무, 길, 사람 같은 전경을 함께 넣어보세요. 사진에 깊이감이 생깁니다.
- 너무 넓게 담으려 하기보다 강조할 대상을 하나 정하세요. 사진의 중심이 분명해집니다.
- 역광이 강하면 화면에서 하늘을 눌러 밝기를 낮추세요. 하늘 디테일이 더 잘 남습니다.

## 8. Category: 야경 (`night`)

### Galaxy S25 Ultra (`galaxy_s25_ultra`)

- 야간 모드를 사용하세요. 밝기와 디테일을 동시에 살리기 쉽습니다.
- 촬영할 때 손을 고정하세요. 야경은 흔들림이 생기면 전체 선명도가 크게 떨어집니다.
- 밝은 간판이나 조명이 많을 때는 노출을 조금 낮추세요. 번짐이 줄고 분위기가 살아납니다.
- 가능하면 벽, 난간, 테이블에 기대어 찍으세요. 작은 고정만으로 결과가 훨씬 안정됩니다.

### Galaxy S26 Ultra (`galaxy_s26_ultra`)

- 야간 모드를 켜고 촬영 순간에는 잠깐 멈추세요. 흔들림이 줄어 선명한 결과를 얻기 쉽습니다.
- 밝은 조명을 직접 크게 담기보다 주변 거리 분위기와 함께 담아보세요.
- 네온이나 간판이 강하면 노출을 살짝 낮추세요. 색 번짐이 줄고 글자가 더 또렷해집니다.
- 3배 줌은 어두운 곳에서 흔들림이 커질 수 있습니다. 가능하면 기본 배율부터 안정적으로 찍으세요.

### 일반 스마트폰 (`generic_smartphone`)

- 어두운 곳에서는 최대한 흔들리지 않게 고정하세요. 손떨림만 줄여도 사진이 많이 좋아집니다.
- 밝은 빛을 정면으로 크게 찍기보다 주변 풍경과 함께 담으세요. 분위기가 더 자연스럽습니다.
- 너무 확대해서 찍으면 노이즈가 심해질 수 있습니다. 기본 배율을 먼저 사용하세요.
- 사람이 함께 나온다면 가로등이나 밝은 조명 가까이 세워보세요. 얼굴이 더 자연스럽게 보입니다.

## 9. Category: 자동차/액션 (`automotive_action`)

### Galaxy S25 Ultra (`galaxy_s25_ultra`)

- 3배 줌으로 조금 떨어져 찍으세요. 차량 비율이 안정되고 배경 압축감이 생깁니다.
- 차가 움직이는 방향 쪽에 여백을 남기세요. 속도감이 더 강하게 보입니다.
- 연속 촬영을 사용하세요. 빠른 장면은 여러 장 중 가장 선명한 컷을 고르는 편이 좋습니다.
- 연기나 조명이 강한 장면에서는 노출을 살짝 낮추세요. 차체 디테일이 더 잘 남습니다.

### Galaxy S26 Ultra (`galaxy_s26_ultra`)

- 차량이 화면을 꽉 채우기보다 진행 방향 쪽에 여백을 남기세요. 움직임이 더 자연스럽게 느껴집니다.
- 3배 줌으로 거리를 두면 차체 왜곡이 줄고 배경이 더 단단하게 잡힙니다.
- 드리프트나 빠른 움직임은 연속 촬영으로 잡으세요. 순간 선택의 여지가 훨씬 커집니다.
- 어두운 트랙이나 실내 경기장에서는 노출을 조금 올리되, 조명 번짐이 심하지 않은지 확인하세요.

### 일반 스마트폰 (`generic_smartphone`)

- 차량을 화면에 꽉 채우기보다 움직이는 방향에 여백을 남기세요.
- 너무 가까이서 찍으면 차체가 왜곡될 수 있습니다. 조금 떨어져서 찍는 편이 안정적입니다.
- 빠른 움직임은 한 장에 기대지 말고 여러 장을 찍은 뒤 가장 선명한 컷을 고르세요.
- 연기나 먼지가 많을 때는 차체가 보이는 순간을 기다리세요. 주제가 더 또렷하게 살아납니다.

## 10. 사용 방식

사전 촬영 가이드는 아래 입력만 사용한다.

```json
{
  "category": "automotive_action",
  "deviceType": "galaxy_s25_ultra"
}
```

응답은 아래처럼 3~5개 카드로 구성한다.

```json
{
  "category": "automotive_action",
  "deviceType": "galaxy_s25_ultra",
  "tips": [
    {
      "title": "진행 방향에 여백 남기기",
      "body": "차가 움직이는 방향 쪽에 여백을 남기세요. 속도감이 더 강하게 보입니다."
    }
  ]
}
```

## 11. 금지 사항

- 업로드 이미지 요구 금지
- 분석 결과 요구 금지
- `photoId` 요구 금지
- `analysisId` 요구 금지
- DB 저장 금지
- 검색 기능 추가 금지
- 즐겨찾기 추가 금지
- 1차 범위 밖 카테고리 추가 금지

## 12. 향후 확장 후보

1차 베타 이후 사용자 반응을 보고 아래 기능을 검토한다.

- 여행/한옥
- 봄/꽃
- 실내/카페
- 반려동물
- 제품 사진
- 검색
- 즐겨찾기
- 사용자별 기기 저장
- 실제 EXIF 기반 기기 자동 감지
