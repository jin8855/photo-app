# Beta Test Guide And Feedback Checklist

## 목적

베타 사용자에게 서비스를 설명 없이 사용하게 하고,
어디서 막히는지, 무엇이 헷갈리는지 확인한다.

## 사용자 안내 문구

아래 문구를 그대로 복사해서 사용할 수 있다.

```text
사진 한 장으로 감성 캡션과 이미지를 만들어보세요.

사용 방법:
1. 사진을 올립니다
2. 분석하기를 누릅니다
3. 결과를 확인합니다
4. 직접 꾸미기로 원하는 스타일을 적용합니다
5. 결과 이미지를 만들고 저장하거나 캡션을 복사합니다

설명 없이 한 번 사용해보고,
불편한 점이나 헷갈리는 부분을 자유롭게 알려주세요.
```

## 테스트 시나리오

사용자에게는 아래 항목을 직접 말하지 않는다.

- 홈 진입
- 업로드
- 분석
- 결과 확인
- 직접 꾸미기 진입
- 프리셋 선택
- 결과 이미지 만들기
- 저장 또는 복사

## 관찰 포인트

### 멈춤

- 어디에서 멈추는가
- 몇 초 이상 고민하는가

### 질문

- 무엇을 물어보는가
- 어떤 표현을 이해하지 못하는가

### 오해

- 잘못된 버튼을 누르는가
- 흐름을 반대로 이해하는가

## 체크리스트

### 홈

- 바로 무엇을 해야 할지 알 수 있는가

### 업로드

- 사진 올리기가 자연스러운가

### 분석

- 분석하기 의미를 이해하는가

### 결과 화면

- 바로 쓸 수 있는 영역을 찾는가
- 분위기, 말투, 감정 차이를 이해하는가

### 직접 꾸미기

- 기본 꾸미기만으로 충분히 사용 가능한가
- 자세히 꾸미기를 굳이 열지 않아도 되는가

### 결과 생성

- 결과 이미지 만들기 의미를 이해하는가
- 미리보기와 최종 이미지 차이를 이해하는가

### 완료 후

- 저장 버튼을 바로 찾는가
- 복사 버튼을 바로 찾는가

## 피드백 질문

사용 후 아래 질문을 사용자에게 물어본다.

- 어디가 제일 헷갈렸나요?
- 어떤 부분에서 멈췄나요?
- 마음에 들었던 기능은 무엇인가요?
- 이걸 실제로 쓸 것 같나요?
- 뭐가 있으면 더 좋을 것 같나요?

## 목표 기준

설명 없이 끝까지 사용할 수 있는가.

## 운영 팁

- 설명하지 말 것
- 옆에서 지켜보기만 할 것
- 질문이 나오면 기록 먼저 하고, 답변은 나중에 할 것

## 다음 단계

베타 테스트 후에는 공통으로 나온 문제 3개만 수정하고, 나머지는 보류한다.
