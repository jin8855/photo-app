# STEP 2 - TONE DEFINITIONS

## 목적
톤별 결과 차이를 명확하게 정의

---

## Tone 목록

### warm
노란톤 / 부드러움 / 낮은 대비

### vivid
높은 채도 / 높은 대비 / 강한 느낌

### calm
회색톤 / 낮은 채도 / 고요함

### film
바랜 색감 / grain / 날짜 / 필름 프레임

### legacy_digital
푸른톤 / 낮은 대비 / 디지털 감성 / 날짜

### black_white
색 제거 / 대비 강조 / 감정 집중

### focus_subject
배경 흑백 / 피사체 컬러 유지

---

## 규칙

- 톤은 눈으로 바로 차이가 나야 한다
- 설명 없이 이해 가능해야 한다
