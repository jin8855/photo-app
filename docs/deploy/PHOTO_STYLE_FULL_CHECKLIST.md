# PHOTO_STYLE_FULL_CHECKLIST.md

## 목적
현재 사진 스타일 기능이 제대로 구현되었는지 점검하기 위한 체크리스트

---

## 핵심 원칙
스타일 = 색감 + 프레임 + 오버레이 + 배경 처리

---

## 스타일 체크리스트

### 1. 기본
- [ ] 원본 유지
- [ ] 변화 없음 확인

---

### 2. 흑백
- [ ] 전체 흑백 처리
- [ ] 대비 조정

---

### 3. 필름
- [ ] 필름톤 색감 적용
- [ ] 프레임 적용
- [ ] 날짜 오버레이 적용
- [ ] 미리보기 반영
- [ ] 결과 이미지 반영

---

### 4. 빈티지
- [ ] 색감 저채도
- [ ] 노이즈(그레인)
- [ ] 따뜻한 톤

---

### 5. 인물 강조
- [ ] 인물 컬러 유지
- [ ] 배경 흑백/저채도
- [ ] 중심 강조
- [ ] 미리보기 반영

---

### 6. 공통
- [ ] 클릭 시 즉시 반영
- [ ] 선택 상태 표시
- [ ] 결과 이미지 동일 적용

---

## PASS 기준
- 모든 스타일이 시각적으로 차이가 명확함
- 클릭 시 즉시 반응

---

## HOLD 기준
- 클릭해도 변화 없음
- 스타일 간 차이 없음
