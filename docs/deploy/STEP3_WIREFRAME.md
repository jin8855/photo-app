## 촬영 가이드

상황 선택 → 대상 선택 → 기기 선택

결과:
- 빠른 팁 3개
- 주의사항
- 설정 툴팁

---

## 사진 분석

사진 업로드 → 대표 선택

---

## 결과 화면

1. 대표 이미지
2. 패키지
3. 톤 선택
4. 스타일 선택
5. 테마 선택
6. 캡션 3개
7. 이미지 생성
8. 사진 개선 팁
9. 직접 꾸미기

---

## 직접 꾸미기

- 텍스트 위치
- 날짜 ON/OFF
- 프레임 변경
- 밝기/대비/채도

---

## 핵심 UX

- 선택 3개 이하
- 클릭 1번 결과
- 기본값 완성
