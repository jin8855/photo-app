# FREE_SAMPLE_PHOTO_SET_AND_AI_PROMPT_PACK.md

## 목적
촬영 가이드를 더 직관적으로 만들기 위해
- 무료로 바로 구할 수 있는 샘플 사진 검색 세트
- AI 생성 샘플 이미지 프롬프트 세트
- 보고 따라 찍을 때의 핵심 체크포인트

를 한 번에 정리한다.

---

# 1. 추천 사용 방식

## A. 무료 샘플 사진
- 실제 구도 / 빛 / 거리감을 참고할 때 사용
- 촬영 가이드 예시 이미지로 연결하기 좋음

## B. AI 생성 샘플 이미지
- 원하는 조건 조합이 무료 사이트에 잘 없을 때 사용
- 노을 + 바다 + 원경, 공연 + 인물 + 중간 같은 특정 장면을 빠르게 시각화할 때 유용

## C. 따라 찍기
- 샘플을 그대로 복제하기보다
- 구도 / 빛 방향 / 거리 / 피사체 배치만 참고

# 2. 무료 샘플 사진 사이트

## Unsplash
- 여행, 풍경, 바다, 꽃, 카페, 음식 참고용으로 좋음
- 검색 키워드가 자연어여도 결과가 잘 나옴

## Pexels
- 인물, 음식, 카페, 제품, 공연/야간 계열 참고용으로 좋음
- 세로형 / 쇼츠형 이미지도 찾기 쉬움

## Pixabay
- 풍경, 자동차, 꽃, 날씨, 하늘 계열 참고용으로 좋음
- 비교적 넓은 주제 검색에 강함

# 5. 보고 따라 찍을 때 체크포인트

## 1. 거리부터 맞추기
- 접사인지
- 중간인지
- 원경인지

이걸 먼저 맞추면 사진 느낌이 가장 빨리 비슷해진다.

## 2. 빛 방향 보기
- 정면광인지
- 옆빛인지
- 역광인지

빛 방향이 다르면 같은 대상도 완전히 다르게 보인다.

## 3. 배경 비율 보기
- 배경을 얼마나 남겼는지
- 하늘 / 바닥 / 테이블 / 벽 비중이 어떤지

## 4. 수평 / 대각선 확인
- 바다 / 산 / 거리 / 자동차는 구도선이 중요하다

## 5. 색감보다 구조 먼저
- 처음엔 필터보다
- 구도 / 거리 / 빛 / 배치부터 따라가기

# 8. Codex 실행 프롬프트

촬영 가이드 예시 이미지를 더 직관적으로 만들기 위해
무료 샘플 사진 세트와 AI 생성 샘플 세트를 참고 구조로 반영해.

요구:
1. exampleImages 구조 유지
2. 무료 샘플 사진은 위 검색 키워드 기준으로 교체 가능하게 경로 구조 유지
3. AI 생성 샘플은 실제 이미지가 없을 때 참고용으로 사용할 수 있게 정리
4. UI에서는 1~2장만 보여주고, 자세히 보기에서 추가 설명 연결

중요:
- 예시 이미지는 참고용이다
- 실제 촬영 시 따라 찍기 체크포인트를 같이 보여주면 좋다

결과:
1. exampleImages 반영 방식
2. placeholder 교체 우선순위
